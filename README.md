![Django](https://img.shields.io/badge/Django-4.0+-092E20?style=flat&logo=django&logoColor=white)
![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS-38B2AC?style=flat&logo=tailwind-css&logoColor=white)

## סקירה כללית

פלטפורמה מקיפה לניהול משימות שנועדה לייעל את שיתוף הפעולה בצוותים ולספק מעקב יעיל אחר פרויקטים. המערכת כוללת ממשק משתמש מודרני בסגנון "Dark Neon" בעל ניגודיות גבוהה שנבנה באמצעות Tailwind CSS, המבטיח חווית משתמש ממוקדת ויעילה.

האפליקציה מיישמת בקרת גישה מבוססת תפקידים (RBAC) קפדנית, המבדילה בין מנהלי מערכת למשתמשים רגילים כדי לשמור על שלמות הנתונים והלוגיקה העסקית.

## תכונות מרכזיות

### ממשק וחווית משתמש
* **עיצוב מודרני:** ממשק Dark Mode מותאם אישית המשתמש בהדגשות צבע ניאון ליצירת היררכיה ויזואלית ברורה.
* **רספונסיביות:** תצוגה מותאמת באופן מלא למסכי מחשב ומכשירים ניידים.
* **אלמנטים אינטראקטיביים:** סינון דינמי ועדכוני סטטוס ללא צורך ברענון עמוד מיותר.

### הרשאות ותפקידים
המערכת אוכפת הפרדת תפקידים ברורה:

1.  **מנהל מערכת (Admin):**
    * יכולת ליצירת משימות חדשות ושיוכן לצוותים.
    * תצוגה גלובלית של כל המשימות במערכת.
    * הרשאה לערוך או למחוק משימות (מוגבל למשימות שטרם שויכו לעובד, למניעת שיבוש בזרם העבודה).

2.  **חבר צוות (User):**
    * צפייה במשימות המשויכות לצוות הספציפי שלו.
    * פונקציונליות של "שיוך עצמי" (Claim) למשימות חדשות.
    * ניהול סטטוסים (סימון משימות כ"בתהליך" או "הושלם").
    * סינון מותאם אישית (תצוגת "המשימות שלי").

### פונקציונליות טכנית
* **סינון דינמי:** סינון בצד שרת (Server-side) לפי סטטוס המשימה ושיוך למשתמש.
* **ולידציית נתונים:** לוגיקה למניעת בחירת תאריכי עבר והבטחת תקינות שדות החובה.
* **אבטחה:** שימוש במערכת האימות המובנית של Django והגנת CSRF בטפסים.

## הוראות התקנה והרצה

עקוב אחר הצעדים הבאים להרצת הפרויקט בסביבה מקומית:

1.  **שכפול המאגר (Clone):**
    ```bash
    git clone [https://github.com/sarisholomon/task_manager.git](https://github.com/sarisholomon/task_manager.git)
    cd task_manager
    ```

2.  **התקנת תלויות:**
    מומלץ להשתמש בסביבה וירטואלית (Virtual Environment).
    ```bash
    pip install -r requirements.txt
    ```

3.  **הרצת מיגרציות למסד הנתונים:**
    ```bash
    python manage.py migrate
    ```

4.  **הפעלת שרת הפיתוח:**
    ```bash
    python manage.py runserver
    ```

5.  **כניסה למערכת:**
    פתח את הדפדפן וגלוש לכתובת: `http://127.0.0.1:8000`.
